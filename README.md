## Overview
Autotrace is a work-in-progress web application built as a learning project, focused on designing and implementing a modern, scalable frontend architecture using the latest React ecosystem.

The goal of this project is to practice real-world patterns commonly used in production-grade SaaS applications, including authentication, routing, state management, and form handling.

### Tech stack
- **Frontend:** React 19, TypeScript, Vite
- **Routing:** TanStack Router
- **State Management:** Zustand, TanStack React Query
- **Forms & Validation:** React Hook Form, Yup
- **Authentication & Backend:** Supabase
- **UI & Styling:** Tailwind CSS, Radix UI, Storybook
- **Testing:** Vitest, React Testing Library
- **Tooling:** ESLint, Prettier
